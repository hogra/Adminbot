Товарищ Админ
Бот позволяет управлять чатом не являясь организатором. Если чат очень большой это удобно, т.к. позволяет 
назначить более 10 таких "администраторов".

У бота есть база данных, состоящая из двух таблиц. Users и Group, соответственно - для пользователя и для 
чатов.

В Groups сохраняются запрещеные в часте слова, максимально количество предупреждений, приветственное 
сообщение, запрет ссылок. Все можно настроить в самом чате.

В Users сохраняется информация о каждом пользователе. Его id в вк, чат в котором он находится, его статус, 
количество символов, дату вступления в чат, количество предупреждений

Перед началом работы бота нужно пригласить в чат (через кнопку в сообществе - https://vk.com/public219582896), 
дать ему доступ ко всем сообщениям и дать права администратора.

Управление ботом осуществляется через команды в чате. Команды начинаются с "!", команды могут использовать 
только админы (за исключением "!топ", "!стат" и "!даты"). Команды нужно писать в ответ на сообщение человека, 
на котором предполагается использование.Ниже приведен список команд.
1. !Админ
	Даёт пользователю статус админа (относительно бота), после этого он может использовать команды.
2. !Снять
	Можно использовать только на админах, использовать может только организатор. Забирает у пользователя 
	статус админа
3. !Кик
	Выгоняет пользователя из чата. Нельзя использовать на админах (если не использует организатор)
4. !-
	Удаляет сообщение пользователя.
5. !Мут {время}
	Дает пользователю статус "мут" и снимает с администратора. Время обязательно должно указываться. На 
	это время с момента использования все сообщения пользователя удаляются. 
		Пример формата времени [10 минут / 10 часов / 10 дней]. 
		НЕ [10 часов 20 минут]
6. !Размут
	Если пользователь в муте - возвращает ему статус "пользователь"
7. !Пред
	Пред/Пред+ 
		Выдает пользователю одно предупреждение. При достижении максимального количества предупреждений, 
		пользователь кикается.
	Пред-
		Снимает одно предупреждение с пользователя. В минут уйти нельзя
8. !Всепрощение
	Снимает все предупреждения со всех пользователей чата
9. !Ключ [настройка] [значение]
	Позволяет изменять настройки чата. Ниже приведен список настроек.
		1. Слово
			Добавляет указанное слово в список запрещенных слов. Если это слово уже в нем - 
			убирает его из списка.
			Сообщения, содержащие запрещенное слово, автоматически удаляются
		2. Ссылки
			Запрещает/разрешает использование ссылок (в зависимости от того, были ли они разрешены).
			Значение не требуется.
		3. Приветствие
			Задает приветственное сообщение. Это сообщение бот будет писать каждый раз, когда в 
			чат приходит новый участник
		4. Преды
			Задает максимальное количество предупреждений. Должно быть больше нуля
10. !Стат
	Если отправлено в ответ на сообщение - показывает статистику указанного пользователя
	Иначе - собственную статистику
	Статистика содержит значения таблицы Users
11. !Топ
	Выводит 10 (или меньше) участников чата по символам.
12. !Даты
	Выводит 30 (или меньше) участников чата по дате вступления в чат.
